Pokédex
This Pokédex was created by Alan Morel using PokéAPI for data. The backend is written in Python using Flask, PIL, and Requests. The frontend is written in HTML5, Javascript and CSS 3. This was created as a dual-class final project for Drawing on the Web and Python for Applications. Pokémon is copyright The Pokémon Company and Nintendo.
